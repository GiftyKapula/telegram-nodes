# Telegram Nodes

This is a concept app for the Telegram Design Contest 2025, implementing "Telegram Nodes".

## Features

- **Main Telegram Interface**: Replicated chat list.
- **Nodes**: Separate workspaces with channels and categories.
- **Drawer Navigation**: Switch between Main Account and Nodes.
- **Profile Customization**: Set different profiles for each Node.
- **Themes**: Day and Night mode support.

## Getting Started

1. Install dependencies:

   ```bash
   npm install
   ```

2. Start the app:

   ```bash
   npm run start
   ```

3. Run on Android/iOS:
   - Press `a` for Android emulator.
   - Press `i` for iOS simulator.
   - Scan QR code with Expo Go app on physical device.
